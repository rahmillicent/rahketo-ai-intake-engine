RahKeto AI Intake & Protocol Assignment Engine
Purpose
This project demonstrates how AI can safely convert wellness intent into a structured ketogenic fasting protocol without medical claims, hallucinations, or unsafe guidance.
What It Does
Collects user goals, fasting experience, and constraints
Assigns a RahKeto Protocol Tier
Outputs a structured plan with guardrails
Why It Matters This is the foundation for personalization, retention, and safety across the RahKeto AI Membership App.
